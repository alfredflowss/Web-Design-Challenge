Summary: Lattitude Vs. X

The purpose of this project was to analyze how the weather changes as changes in that Latitude and Longitude happen based on a city location. To further refine my input dataset, I made use of the Citipy library in python to import the nearest city to the Longitude and Latitude provided in the file city data input file.

My next step was to access the Open weather API via a get request in my jupyter notebook. I retrieved some weather information e.g. (Humidity, cloudiness etc.) for each city in my data set. This site shows the data Visualizations I created as part of the analysis and explain/describes any correlation I was able to uncover.
